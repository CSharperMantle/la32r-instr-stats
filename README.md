# la32r-instr-stats

Web app for collecting statistics over LoongArch32 Reduced binaries.

## Build

### Requirements

* A working Rust toolchain
* [`wasm-pack`](https://rustwasm.github.io/wasm-pack/installer/)
* [`yarn`](https://yarnpkg.com/getting-started/install)

### Commands

```sh
yarn
yarn build
```
